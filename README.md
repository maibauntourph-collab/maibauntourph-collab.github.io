# Portfolio Project

Author: Kwangseob Park

## Project Overview

This is a modern, single-page portfolio website built with HTML, CSS (Tailwind + Custom), and JavaScript. It showcases projects, technical skills, and provides a way to contact the developer.

## Tech Stack

-   **HTML5**: Semantic structure.
-   **CSS**:
    -   **Tailwind CSS**: Utility classes for rapid development.
    -   **Custom CSS**: Specific animations and advanced styling (Neo-Brutalist elements).
-   **JavaScript**: Vanilla JS for interactivity (smooth scrolling, observers, parallax).
-   **Vite**: Fast development server and bundler.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## File Structure

-   `index.html`: Main entry point containing content and styles.
-   `style.css`: Tailwind directives and global styles.
-   `profile.jpg`: Profile image asset.
-   `tailwind.config.js`: Tailwind configuration.
-   `postcss.config.js`: PostCSS configuration.
